Requisitos:

    - Windows
        * Python 3.6 ou mais recente
    - Linux (debian like)
        * Python 3.6 ou mais recente
        * python3-venv

Preparacao do ambiente

comandos(windows):
cd projeto01
python -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt

comandos(linux):
cd projeto01
sudo apt-get install python3-venv python3-tk
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
